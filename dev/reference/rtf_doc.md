# Create an RTF document object

Creation of the object representing an RTF document which can then
receive contents with the
[`rtf_add()`](https://davidgohel.github.io/officer/dev/reference/rtf_add.md)
function and be written to a file with the `print(x, target="doc.rtf")`
function.

## Usage

``` r
rtf_doc(
  def_sec = prop_section(),
  normal_par = fp_par(),
  normal_chunk = fp_text(font.family = "Arial", font.size = 11)
)
```

## Arguments

- def_sec:

  a
  [prop_section](https://davidgohel.github.io/officer/dev/reference/prop_section.md)
  object used to define the default section (page size, margins, header
  / footer, columns) applied to content added before any explicit
  [`block_section()`](https://davidgohel.github.io/officer/dev/reference/block_section.md).

- normal_par:

  an object generated by
  [`fp_par()`](https://davidgohel.github.io/officer/dev/reference/fp_par.md)

- normal_chunk:

  an object generated by
  [`fp_text()`](https://davidgohel.github.io/officer/dev/reference/fp_text.md)

## Value

an object of class `rtf` representing an empty RTF document.

## See also

[`read_docx()`](https://davidgohel.github.io/officer/dev/reference/read_docx.md),
[`print.rtf()`](https://davidgohel.github.io/officer/dev/reference/print.rtf.md),
[`rtf_add()`](https://davidgohel.github.io/officer/dev/reference/rtf_add.md).
See
[`?rtf_add`](https://davidgohel.github.io/officer/dev/reference/rtf_add.md)
for a complete multi-section example exercising `def_sec`, headers /
footers and several
[`block_section()`](https://davidgohel.github.io/officer/dev/reference/block_section.md)
calls.

## Examples

``` r
rtf_doc(normal_par = fp_par(padding = 3))
#> rtf document with 0 element(s)
```
