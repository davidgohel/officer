# transform an xml string with images references

This function is useless now, as the processing of images is automated
when using
[`print.rdocx()`](https://davidgohel.github.io/officer/dev/reference/print.rdocx.md).

## Usage

``` r
wml_link_images(x, str)
```

## Arguments

- x:

  an rdocx object

- str:

  wml string
