# Create a list item

Wrap an
[`fpar()`](https://davidgohel.github.io/officer/dev/reference/fpar.md)
with a hierarchy level for use inside
[`block_list_items()`](https://davidgohel.github.io/officer/dev/reference/block_list_items.md).

## Usage

``` r
list_item(x, level = 1L)
```

## Arguments

- x:

  an
  [`fpar()`](https://davidgohel.github.io/officer/dev/reference/fpar.md)
  object or a character string (automatically converted to
  [`fpar()`](https://davidgohel.github.io/officer/dev/reference/fpar.md))

- level:

  hierarchy level, integer starting at 1
