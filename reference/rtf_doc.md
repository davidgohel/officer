# Create an RTF document object

Creation of the object representing an RTF document which can then
receive contents with the
[`rtf_add()`](https://davidgohel.github.io/officer/reference/rtf_add.md)
function and be written to a file with the `print(x, target="doc.rtf")`
function.

## Usage

``` r
rtf_doc(
  def_sec = prop_section(),
  normal_par = fp_par(),
  normal_chunk = fp_text(font.family = "Arial", font.size = 11)
)
```

## Arguments

- def_sec:

  a
  [block_section](https://davidgohel.github.io/officer/reference/block_section.md)
  object used to defined default section.

- normal_par:

  an object generated by
  [`fp_par()`](https://davidgohel.github.io/officer/reference/fp_par.md)

- normal_chunk:

  an object generated by
  [`fp_text()`](https://davidgohel.github.io/officer/reference/fp_text.md)

## Value

an object of class `rtf` representing an empty RTF document.

## See also

[`read_docx()`](https://davidgohel.github.io/officer/reference/read_docx.md),
[`print.rtf()`](https://davidgohel.github.io/officer/reference/print.rtf.md),
[`rtf_add()`](https://davidgohel.github.io/officer/reference/rtf_add.md)

## Examples

``` r
rtf_doc(normal_par = fp_par(padding = 3))
#> rtf document with 0 element(s)
```
